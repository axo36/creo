d’Utilisation

Les règles d’utilisation du service Creo. Merci de les lire attentivement.

1. Objet

Les présentes Conditions Générales d'utilisation (CGU) ont pour objet de définir de manière détaillée les modalités, conditions et règles d'utilisation du site web Creo (ci-après « le Service »), accessible à l'adresse https://axo36.github.io/creo/.

Le Service permet le transfert de données numériques entre deux appareils distincts, via une interface web ou une application, en transitant par une infrastructure serveur intermédiaire.

L'utilisation du Service implique l'acceptation pleine, entière et sans réserve des présentes CGU par tout utilisateur.

2. Description détaillée du fonctionnement

Le Service permet à un utilisateur :

  • de téléverser (envoyer) un fichier depuis un appareil vers un serveur distant,
  • d'associer ce fichier à un code unique généré ou défini par l'utilisateur,
  • de permettre la récupération de ce fichier depuis un autre appareil en saisissant ce même code.

Le système repose sur un mécanisme de correspondance stricte entre :

  • le code fourni lors de l'envoi,
  • et le code utilisé lors de la récupération.

Aucune récupération de fichier ne peut être effectuée sans correspondance valide.

Les données transitent par un serveur utilisant une infrastructure de type Supabase.

La gestion des comptes utilisateurs, de l'authentification et des accès est entièrement assurée par Creo.

Creo conserve également un contrôle technique sur les appareils utilisant le Service, y compris ceux sur lesquels l'application est installée ou le site utilisé, dans le but de :

  • garantir le bon fonctionnement du Service,
  • assurer la sécurité des transferts,
  • prévenir les abus,
  • optimiser les performances.

Ce contrôle peut inclure la collecte de données techniques liées aux appareils, strictement dans le cadre du fonctionnement du Service.

L'utilisateur est seul responsable :

  • de la confidentialité de ses identifiants et codes,
  • de l'usage fait du Service depuis ses appareils,
  • de toute action réalisée via son compte.

4. Données et fonctionnement des transferts

Les données transférées via le Service :

  • sont envoyées vers un serveur distant,
  • peuvent être temporairement stockées,
  • sont accessibles uniquement via un code valide.

Creo met en œuvre des mesures techniques raisonnables pour sécuriser les transferts et le stockage.

Toutefois, l'utilisateur reconnaît expressément que :
