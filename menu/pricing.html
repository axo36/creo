Questions fréquentes

Sur nos tarifs.

Puis-je annuler à tout moment ?+
Oui, sans engagement ni frais. Annulez en un clic depuis votre compte. Vous gardez l'accès jusqu'à la fin de la période payée.
Y a-t-il un essai gratuit du Pro ?+
Le plan gratuit est disponible à vie. Nous préparons une période d'essai Pro de 14 jours — restez à l'écoute.
La facturation est-elle mensuelle ou annuelle ?+
Mensuelle par défaut. Les abonnements annuels sont disponibles avec 2 mois offerts — contactez-nous.
Acceptez-vous les cartes et PayPal ?+
Nous acceptons toutes les cartes via Stripe, ainsi que PayPal et virements SEPA pour les comptes Équipe.
Que se passe-t-il si je dépasse la limite ?+
Sur le plan gratuit, les transferts au-delà de 5 GB sont bloqués. Pas de surcharge — il suffit d'upgrader.
Qu’est‑ce que le code client ?+
Le code client est un code généré dont vous êtes le seul à avoir connaissance. Il permet de récupérer vos fichiers en toute sécurité grâce à ce code unique.
