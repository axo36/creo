Transfert Ultra-Rapide

Creo utilise une architecture P2P pure pour atteindre les limites physiques de votre réseau. Sur LAN 10GbE, attendez-vous à dépasser 2 GB/s de débit soutenu.

  • ✓ Wi-Fi Direct (sans routeur)
  • ✓ LAN câblé 1GbE / 10GbE
  • ✓ Protocole UDP optimisé
  • ✓ Multi-thread parallèle
🔐

Sécurité Zero-Knowledge

Vos données n'appartiennent qu'à vous. Creo n'a jamais accès à vos fichiers ni à vos clés de chiffrement. Tout est calculé localement sur vos appareils.

  • ✓ AES-256-GCM
  • ✓ TLS 1.3 en transit
  • ✓ Clés générées localement
  • ✓ Zéro log de contenu
🌐

Réseau Distant (Pro)

Transférez entre appareils sur des réseaux différents grâce à notre relay chiffré. Traversée NAT automatique, zero configuration requise de votre côté.

  • ✓ Relay chiffré E2E
  • ✓ NAT traversal auto
  • ✓ Tunnel QUIC
  • ✓ Serveurs EU / US / Asia
🔄

Synchronisation (Pro)

Définissez des dossiers synchronisés entre vos appareils. Les changements se propagent en temps réel, avec gestion des conflits et versioning intégré.

  • ✓ Sync bidirectionnel
  • ✓ Résolution de conflits
  • ✓ Historique 30 jours
  • ✓ Filtres d'exclusion
